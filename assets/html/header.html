The header partial of the website is where the main navigation lives. This partial lets you control & customize options related to the header section of the website.

This options can be found in the wordpress admin menu under apperence > customize The header section includes the following information:

  • Logo
  • Website title
  • tagline (slogan)
  • Main menu (primary navigation)

Layout

Setting Field Name Description
Layout Select box *If possible to use images in the select box even better Aengus Aine Banba Eriu Fodla (Default) Bodb Brigid Cliodhna Dagda Danu Dian Etain
In order to avoid complexity naming header templates, each header layout name come from an Irish mythological figure. (Check names here)

Some of the rules for taking names are the following:
  • Only take 1 name (by default the first name)
  • No accents
  • No special characters
