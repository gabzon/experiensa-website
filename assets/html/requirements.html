Agency 2.0 is a web application based on different technologies and Wordpress is at its core. Here is a list of other technologies and dependencies:
Wordpress Bedrock WordPress boilerplate with modern development tools, easier configuration, and an improved folder structure
Theme Sage WordPress starter theme with a modern front-end development workflow.
Plugins Piklist Piklist is a fast and concise plugin that encourages WordPress rapid development. It simplifies many of the more difficult tasks in WordPress and adds functionality not presently common in WordPress core.
WPML WPML makes it easy to build multilingual sites and run them. It’s powerful enough for corporate sites, yet simple for blogs.
WP-API Access your WordPress site’s data through an easy-to-use HTTP REST API
Enhanced Media Library Best way to categorize media library
WP Pusher Deploy WordPress themes and plugins directly from GitHub
Google Captcha (reCAPTCHA) Allows implement security Captcha into web forms.
JetPack Jetpack is a free WordPress plugin that allows you to focus on what you do best. Create and publish great content. We’ll take care of the rest.
Auth0 WordPress Login Plugin - Pain Free. Connect WordPress to every login system on Earth. Really.
